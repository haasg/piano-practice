Major Scale Fingerings — All 12 Keys

One octave, ascending. Descending uses the same fingers in reverse. Print this and keep it on the music stand.

Three rules generate almost everything below.
1. The thumb never plays a black key (in scales).
2. Ascending RH: the thumb tucks under immediately after finger 3 (and the run ends 4–5 at the top).
3. The left hand mirrors the right: LH descending feels like RH ascending.

Family 2 — near-twins of the home fingering

B major keeps the home RH; F major keeps the home LH. Each changes only one hand (B: LH starts on 4; F: RH uses 4 on B♭ so the thumb stays off the black key).

Family 3 — black-key tonics

Start on a long finger; the thumb waits for the first white key. Same logic in every key.

Fingerings as printed in the source PDF (single-octave form, ending LH 2 on top). When playing multiple octaves, the top-note finger is replaced by the loop continuing — e.g. B♭ RH runs 2-1-2-3-1-2-3-4 continuously, and some exam books print the single octave starting RH 4 or 3. Don't worry about that variance until these keys come up in a lesson.
